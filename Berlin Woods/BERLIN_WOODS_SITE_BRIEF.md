# Berlin Woods Treehouses Standalone Website Brief

## Project Summary

Create a standalone website for Berlin Woods Treehouses at `www.berlinwoodstreehouses.com`.

The site should:

- Match the branding and overall feel of `www.amishcountrylodging.com`
- Reuse content and photos from the existing Berlin Woods pages on Amish Country Lodging
- Link directly to the existing booking engine
- Focus on increasing direct bookings
- Improve SEO for Berlin Woods-specific searches
- Keep contact options simple with phone-only calls to action

This project will be built locally first, reviewed, refined, and launched later when ready.

## Confirmed Decisions

- Domain: `www.berlinwoodstreehouses.com` has already been purchased
- Build approach: local development first, launch later
- Booking flow: direct links to the booking engine
- Branding: consistent with Amish Country Lodging
- Photos: use existing Amish Country Lodging Berlin Woods photos
- Content: match the main site
- Main goals: more direct bookings and better SEO
- Contact method: phone only for now

## Business Goals

### Primary Goals

1. Increase direct bookings for Berlin Woods Treehouses
2. Rank better for Berlin Woods and Ohio treehouse lodging searches

### Secondary Goals

1. Give Berlin Woods its own clear identity in search and marketing
2. Make it easier for guests to understand the property quickly
3. Reduce friction from discovery to booking
4. Build trust fast with strong visuals and proof points

## Recommended Website Strategy

The standalone site should be simpler and more conversion-focused than a multi-property lodging website.

That means:

- Fewer navigation choices
- Stronger property-specific messaging
- Direct booking links throughout the site
- Large, high-quality photography
- Clear explanations of the different treehouse options
- SEO content targeted specifically to Berlin Woods, Berlin, Ohio, Amish Country, and treehouse lodging

## SEO Strategy

### Core SEO Targets

The site should be structured to rank for searches like:

- Berlin Woods Treehouses
- Berlin Ohio treehouses
- treehouse lodging in Ohio
- Amish Country treehouses
- romantic getaways in Berlin Ohio
- unique lodging in Amish Country
- treehouse rentals near Berlin Ohio

### SEO Recommendations

- Use one clear H1 per page
- Write unique title tags and meta descriptions for each page
- Include internal links between Home, Treehouses, Gallery, FAQ, and Location
- Use descriptive image alt text
- Include property-specific copy instead of generic lodging language
- Add schema markup later if possible for lodging/business data
- Ensure fast mobile performance
- Make booking buttons easy to find above the fold

### Suggested Homepage SEO Direction

Possible title tag:

- `Berlin Woods Treehouses | Treehouse Lodging in Berlin, Ohio`

Possible meta description:

- `Stay at Berlin Woods Treehouses in Berlin, Ohio for a unique Amish Country getaway with fireplaces, hot tubs, full kitchens, and direct booking online.`

## Conversion Strategy

The site should gently but consistently move visitors toward booking.

Recommended conversion patterns:

- Book now button in the header
- Book now button in the hero
- Direct booking button on every unit card
- Repeated booking CTA near the footer
- Mobile-friendly sticky booking button if it feels natural in design
- Phone number always visible

## Design Direction

The website should visually align with Amish Country Lodging while feeling focused on one special property.

Recommended design cues:

- Warm, natural, welcoming palette
- Rustic-luxury photography
- Clear typography with easy reading
- Mobile-first layout
- Large image sections
- Strong emphasis on wooded setting and elevation/treehouse experience

Because the site is standalone, it should feel more premium and less crowded than the parent multi-property site.

## Technical Recommendation

For a first standalone site, the best approach is a simple modern static website.

Recommended build approach:

- A lightweight front-end site with fast loading pages
- Clean SEO-friendly HTML structure
- Reusable components for header, footer, gallery, and unit cards
- Direct external links to the existing booking engine
- Easy hosting later on a modern platform

Good implementation options:

- Plain HTML/CSS/JavaScript for maximum simplicity
- Or a lightweight framework if we want cleaner scaling

Recommendation for this project:

- Start with a fast, simple static site build that is easy to understand and easy to launch
